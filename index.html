About — 요약

Python, SQL, Excel, Tableau를 활용하여 데이터 수집·전처리·분석·시각화 전 과정을 수행한 신입 데이터 분석가입니다. 아래 프로젝트에서 문제 정의 → 접근 방식 → 핵심 결과 → 비즈니스 제안을 채용 담당자 관점으로 요약했습니다.

목표

데이터 기반 의사결정 지원과 실무에 기여할 분석 역량 확보

Tech Stack

Python

Pandas, NumPy, Matplotlib, Seaborn — 데이터 처리, 시각화

SQL

MySQL, PostgreSQL — 데이터 쿼리, 집계, 조인

Excel

피벗, 함수, 차트 — 빠른 탐색적 분석

Tableau

대시보드 제작, 스토리텔링 시각화
